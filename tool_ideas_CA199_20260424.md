# 研究ツール案: CA19-9 glycan antibody (2026-04-24)

## 課題分析と優先順位付け
ボトルネック分析から、以下の課題が抽出されました。
1. ルイス陰性患者でのCA19-9非産生（代替マーカーの必要性）
2. 正常ルイス抗原への交差反応性と特異性の低さ
3. 膵がん特有の厚い間質による抗体の浸潤阻害

これらのうち、インシリコでの予測・最適化による解決価値が高く、かつPython/Streamlitベースで中〜低コストで迅速にプロトタイプ化できる領域を優先しました。

## 提案ツール案（3選）

### 案1: CA19-9 Cross-Reactivity Predictor (SBDDベース交差反応性予測ツール) 【最有力】
- **解決する課題**: 正常組織における類似ルイス抗原との交差反応性・オンターゲット毒性リスクを設計段階でスクリーニングする。
- **機能概要**: 抗体の配列や構造を入力とし、CA19-9と類似ルイス抗原（Le^a, sLe^x等）への結合親和性をIn silicoで比較スコアリング・可視化する。
- **実装コスト**: 中
- **ポートフォリオ連携**: `GlycoAntibodyStudio` と統合し、糖鎖特異的な結合親和性評価のモジュールとして活用可能。物理化学的性質の算出には `Protein Hydrophobicity Profiler` のロジックを流用できる。

### 案2: Lewis-Negative Biomarker Imputation Tool (ルイス陰性患者向け代替マーカー予測)
- **解決する課題**: ルイス陰性患者における、CA19-9に代わる最適な代替バイオマーカー（CD44等）のパネルを予測・提示する。
- **機能概要**: 患者のRNA-seq発現データまたは簡易パネルデータを入力し、CA19-9陰性背景における腫瘍進行度や予後をAIモデルが推定し、最適なマーカーを提示する。
- **実装コスト**: 低〜中
- **ポートフォリオ連携**: 既存の `RNA-seq DEG解析アプリ` の拡張モジュールとして実装し、発現量変動から患者層を自動分類するワークフローに組み込むことが可能。

### 案3: Pancreatic Stroma Penetration Simulator (膵がん間質浸透性シミュレータ)
- **解決する課題**: 厚い間質に阻まれて腫瘍深部に抗体が到達しない問題のインシリコ予測。
- **機能概要**: 抗体の分子量・等電点・疎水性スコアと、腫瘍組織の細胞外マトリックス(ECM)密度を入力し、間質における拡散係数や到達深度をシミュレーションする。
- **実装コスト**: 中〜高
- **ポートフォリオ連携**: `Tissue-Spatial-Analysis` プラットフォームや `Intrabody-Design-Platform` と連携し、空間トランスクリプトームから得られたECM成分の分布データを利用して局所的な浸透性をマップ化する。

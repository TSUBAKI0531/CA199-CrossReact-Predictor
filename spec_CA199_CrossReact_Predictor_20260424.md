# 仕様書: CA19-9 Cross-Reactivity Predictor

## 1. 概要
CA19-9抗体開発において、正常ルイス抗原（sLe^x, Le^aなど）との交差反応性を予測・スクリーニングするためのStreamlitアプリケーション。

## 2. 目的
- 抗体の配列（FASTA）等から、糖鎖抗原ごとの結合性スコアを比較し、特異性の高いリード抗体を選別する。
- `GlycoAntibodyStudio` や `Protein Hydrophobicity Profiler` のエコシステムの一部として機能させる。

## 3. 主要機能
1. **抗体データの入力**:
   - FASTA配列入力（VH/VL）。
2. **交差反応性スコアリング (Mock)**:
   - BioPythonを用いて配列特性（等電点、分子量、芳香族アミノ酸割合）を抽出。
   - CA19-9 (sLe^a) に対するスコアと、類似抗原 (sLe^x, Le^a, Le^b) に対するバインディングスコアをヒューリスティックベース（プロトタイプ用モック）で算出。
3. **結果の可視化**:
   - PandasとStreamlitを用いた、各抗原への結合プロファイルの棒グラフ化。
   - アラート表示（例: sLe^xへの交差反応リスクが高い場合は警告を表示）。

## 4. 技術スタック
- **言語**: Python 3.9+
- **フレームワーク**: Streamlit
- **ライブラリ**: pandas, biopython, matplotlib (可視化用)

## 5. UI/UXフロー
1. サイドバーで抗体配列を入力。
2. 「予測を実行」ボタンをクリック。
3. メインパネルに抗体の物理化学的プロパティが表示される。
4. 下段にCA19-9と他のルイス抗原に対する「相対結合予測スコア」をグラフ表示し、交差反応リスクをテキストで解説。
